How to Contribute

1

Choose a Project

Browse our repositories and pick a project that interests you or aligns with your skills.

2

Read Guidelines

Check the CONTRIBUTING.md file for setup instructions, coding standards, and contribution guidelines.

3

Fork & Develop

Fork the repository, create a feature branch, and implement your changes or bug fixes.

4

Submit PR

Create a pull request with a clear description of your changes. Our team will review and provide feedback.
